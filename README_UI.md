# Tu Tien Idle - UI Test mới

## Cách chạy

```bash
npm install
node server.js
```

Mở:

```text
http://localhost:3000
```

## Luồng test nhanh

1. Đăng nhập bằng `demo`.
2. Vào tab `Test Dev`.
3. Chọn linh căn, ví dụ `Kim`.
4. Bấm `Cấp dữ liệu test`.
5. Vào tab `Công Pháp` để học/trang bị skill.
6. Vào tab `Đan Dược` để luyện đan/dùng đan.
7. Vào tab `Động Phủ` để xem linh khí, công trình, dược viên.

## Ghi chú

UI này dùng các API test hiện có, chưa phải giao diện cuối cùng. Mục tiêu là test hệ thống mới dễ hơn giao diện cũ.
